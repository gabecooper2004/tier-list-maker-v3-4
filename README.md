## What Tier-List-Maker Does

Tier-List-Maker is a compact, browser-first utility for building personalized ranking boards and tier lists. It targets Windows users who want an easy way to sort items into tiers without installing a traditional desktop program.

Because it runs offline and saves information in local storage, the app is well suited for quick edits and repeated use in the browser. You can also bring in images for items, making it straightforward to assemble visual rankings and export them in a format that works for sharing or keeping an archive.
